ENTREGABLE 1

Creación de la página web

Creación de un repositorio en github

Se creó una cuente grupal en GitHub, en la cual se diseñó un repositorio para el presente proyecto. Asimismo, se configuró GitHub pages para poder publicar el sitio web y Manage Access; para compartir el repositorio con todos los integrantes.

Esquema base

Para elaborar el esquema base de la página se hizo uso del programa Mobirise. Las fuentes utilizadas para el diseño fueron proporcionadas por el programa anteriormente mencionado y Google. Una vez culminada la elaboración del esquema, este se subió al repositorio de GitHub.

Vinculación Mobirise - Github

Además se utilizó el programa PyCharm para realizar la conexión entre GitHub y Mobirise, guardando los respectivos cambios hechos por el equipo.
